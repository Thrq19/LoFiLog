### 1. ðŸ  Dashboard Interaktif
Pusat kendali utama Anda yang cerdas dan personal.
- **Real-time Greeting**: Menyapa Anda sesuai waktu (Pagi/Siang/Malam) dengan nama panggilan yang dapat dikustomisasi.

### 2. ðŸ“ Daily Notes (Jurnal & Catatan)
Tempat untuk mencatat ide, perasaan, atau log harian.
- **Secret Mode ðŸ”’**: Tandai catatan sebagai "Rahasia" agar tidak muncul di daftar utama. Akses catatan tersembunyi melalui ikon gembok di AppBar.
- **Search & History**: Cari catatan berdasarkan judul/isi dan lihat riwayat lengkap.
- **Manajemen Mudah**: Ketuk untuk membaca detail, tombol khusus untuk menghapus.

### 3. â±ï¸ Focus Time (Smart Pomodoro)
Timer fokus yang dirancang agar Anda tidak kehilangan progres.
- **Pilihan Durasi**: Preset cepat (25, 45, 60 menit).
- **Background Persistence**: Timer **tetap berjalan** meskipun aplikasi ditutup.
    - *Logika*: Aplikasi menyimpan timestamp deadline di database. Jika Anda membuka kembali aplikasi sebelum waktu habis, timer berlanjut sinkron. Jika lewat, alarm sesi selesai akan menyambut Anda.

### 4. ðŸ“… To-Do & Calendar Manager
Sistem manajemen ganda untuk **Tugas** dan **Jadwal Kuliah**.

#### Tab Tugas (To-Do)
- **Detail Lengkap**: Judul, Mata Kuliah, Deadline, dan Catatan tambahan.
- **Kalender Visual**: Penanda titik (marker) pada tanggal yang memiliki deadline.
- **Interaksi Cepat**:
    - *Tap Checkbox*: Tandai selesai (tugas menjadi redup).
    - *Long Press*: Hapus tugas secara permanen.
    - *Toggle View*: Sembunyikan/tampilkan tugas yang sudah selesai.

#### Tab Jadwal Kuliah (Schedule)
- **Academic Oriented**: Input detail Mata Kuliah, Hari, Jam, Ruangan, dan Dosen.
- **Organized List**: Jadwal terurut otomatis berdasarkan hari.
